#### IPC Project
Project description @[Progetto Systemcalls.pdf](https://github.com/Xiryl/univr/blob/master/operating-system/ipc/Progetto%20Systemcalls.pdf)

#### Folder Structure
```
  ipc/
  ├── include/
  │   └── *.h 
  ├── src/
  │   └── *.c
  ├── Makefile
  ├── doxygen.cfg
  ├── ipc_clear.sh
  └── file_input.txt
```

#### Run Project
  ` $> ./bin/key_finder <input_file> <output_file>`
  
#### Compile Project
  ` $> make install`
 
#### Create Project DOC
  ` $> make doc`
  
#### Help
  ` $> make help`
